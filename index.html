10 Seconds

A Ludum Dare 27 Entry

10 seconds was greated for Ludum Dare 27, a 48 hour Game Jam. The theme of the competition was the same as my entrie's title, 10 seconds.

This is the first time I've participated in a game jam, and I hope you enjoy it. If you liked the game (or did not) and want to tell me about it, you can do so via Twitter or at the entry page.

Your goal is to move your character through each level to the checkered goal before your time runs out.. To achieve this, you will need to use special items which are either already placed in the level or can be placed by yourself by selecting them in the item bar and then clicking in the level.

This game was created in 48 hours, using

  • Cocos2d-HTML5
  • Coffeescript
  • Google Fonts (Montserrat and Montserat Subrayada)
  • GitHub (For source control and static content hosting)
  • Yeoman
  • Adobe Illustrator
